# Pathfinder Pal

AI-powered career platform that generates adaptive technical assessments 
and personalized learning paths using Google's Gemini 2.5 Flash.

## Features
- Dynamic MCQ and coding exam generation using LLMs
- Context-aware evaluation of student answers
- Personalized micro-learning recommendations
- Real-time exam workflow with Supabase backend

## Tech Stack
Frontend: React + TypeScript + Vite  
Backend: Supabase Edge Functions  
AI Model: Gemini 2.5 Flash  
Database: Supabase PostgreSQL

## AI Engineering Concepts Used
- Prompt Engineering
- Context Injection
- Dynamic Question Generation
- LLM-based Answer Evaluation
- AI-driven Learning Path Optimization

## Architecture
User → React Frontend → Supabase Edge Function → Gemini API → Response → Evaluation
